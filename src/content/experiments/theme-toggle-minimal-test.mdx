This experiment checks whether an icon-only theme switch is readable enough without extra text and whether the interaction still feels obvious in the lab UI.

## Hypothesis

A compact icon-only toggle can stay discoverable if it sits in a stable location and responds clearly on hover.

## What to observe

- whether the control is noticeable in the header/footer
- whether the icon state is clear after switching
- whether the control feels too small on mobile

## Plan

- keep the control icon-only
- test it in both themes
- revise only if the control feels ambiguous in use
